Java中的整数

Java中数值以二进制保存，数的最高位表示符号。
+5表示为
0 00…101
负数一般用补码表示
例如-5为
1 0101
补码则为
1 1011

位运算符

位运算符可以在处理整型变量时，对每一位直接操作。

  • & 位与
    System.out.println(0b1010 & 0b1100);
    二进制结果1000，输出为8
  • | 位或
    System.out.println(0b1010 | 0b1100);
    二进制结果1110，输出结果为14
  • ^ 位异或
    System.out.println(0b1010 & 0b1100);
    二进制结果0110，输出结果为6
  • ~ 位非
    System.out.println(~ 0b0101);
    包括符号位，逐位取反
    二进制结果1010，为1110的补码，输出结果为-6
  • << 左移运算符
    System.out.println(5 << 2);
    除符号位，向左移动两位。二进制结果10100，输出结果为20
  • >> 右移运算符
    System.out.println(-5 >> 2);
    与左移运算符类似。-5表示为1 1011，右移两位是1 1110，属于1 0010的补码，故输出结果为-2
  • >>> 无符号右移运算符
    System.out.println(-5 >>>2);
    类似右移运算符，但是包括了符号位。-5表示为1 1…1011，右移1位，变成了0 11…1101，是一个正数，输出结果变成了2的31次方-3，即2147483645（java中默认一个数是32位）
    不存在<<<无符号左移运算符

字符串相关

Java 中的字符串不可变。修改一个字符串的方法是重新定义。

  • 定义
    String HelloWorld = "Hello, World!";
  • 提取子串
    String hello = String.substring(0,5); //"Hello"
String world = String.substring(7,12); //"World"
  • 串联字符，并用定界符分割
    String All = String.join(",", hello, world, hello); //"Hello,World,Hello"
  • 检测字符串是否相等
    hello = "hello";
System.out.println("Hello".equals(hello)); //区分大小写，false
System.out.println("Hello".equalsIgnoreCase(hello)); //不区分大小写，true
    不要用==判断，==判断的是字符串的位置（类似C++中的指针）
  • 检测空串、Null
          String str = "";
      if(str == null || str.length() == 0)
          System.out.println(1);
      else
          System.out.println(0);
    先检测null，再检测空串
  • 代码单元与码点
    String greeting = "Hello";
int n = greeting.length(); //返回代码单元数量
char first = greeting.charAt(0); //返回第0个代码单元
//得到第3个码点
int index = greeting.offsetByCodePoints(0,3); //offset有位移的意思，这个语句的意思是从0处位移i个点
int cp = greetingcodePointAt(index);
  • 字符串构建
    StringBuilder builder = new StringBuilder();//构建一个空的字符串构建器
builder.append(hello);//添加字符
String newHello = builder.toString();//创建String对象

输入和输出

        Scanner in = new Scanner(System.in);//输入需要先构建一个Scanner对象
        System.out.print("What's your name?");
        String fullName = in.nextLine(); //读取一行
        System.out.print("Tell me again.");
        String firstName = in.next(); //读取第一个单词，以空格为间隔
        System.out.print("How old are you?");
        int age = in.nextInt(); //读取第一个整数
        System.out.print("How tall are you?");
        double height = in.nextDouble(); //读取第一个浮点数
        System.out.print("full name:" + fullName + "\nfirst name:" 
    + firstName + "\nage:" + age + "\nheight:" + height);

第一个输入例子

What's your name?
Dennis Lin
Tell me again.
Dennis
How old are you?
18
How tall are you?
187.5
full name:Dennis LIN
first name:Dennis
age:18
height:187.5

第二个输入例子

What's your name?
Dennis Lin
Tell me again.
Dennis 18 187.4
How old are you?
How tall are you?
full name:Dennis Lin
first name:Dennis
age:18
height:187.4

除nextLine，若一次性输入多项，则会对应到之后的各项输入。
若在Tell me again 之后输入 Dennis Lin，Lin会触发nextInt的错误。
